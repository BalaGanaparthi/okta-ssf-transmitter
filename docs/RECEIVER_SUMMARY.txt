HOW TO USE:

1. Start Receiver:
   $ ./scripts/receiver.sh
   
   Receiver listens on: http://localhost:8081/receive-set

2. Okta Sends SETs to Receiver:
   Configure Okta stream to POST to: /receive-set

3. Watch Console Logs:
   Beautifully formatted boxes show:
   - Event type with icon (👤🌐📱🔒🚪✉️)
   - User email
   - Device ID
   - Timestamp
   - All event fields
   - Reason messages

4. View Event History:
   $ curl http://localhost:8081/events
